# Web Corporativa de FoodLogistic

Aquesta és la web corporativa de FoodLogistic, una empresa fictícia dedicada a la logística alimentària nacional i la distribució integral amb control de la cadena de fred dins el mòdul Projecte Intermodular.

## Demostració

[🌐 URL de la web](https://arnaudominguez.github.io/web/)

## Tecnologies Utilitzades

Per la creació d'aquesta web, s'han utilitzat les següents tecnologies i eines:

- Gemini.
- HTML5/CSS3
- Statcounter per l'analítica
